# 📊 HR Analytics Dashboard

An interactive HR Analytics Dashboard built in Power BI to transform employee data into meaningful workforce insights. The dashboard highlights employee attrition, workforce distribution, job satisfaction, and other HR metrics that support strategic decision-making.

---

## 🚀 Highlights

- Interactive HR dashboard built using Power BI
- Analyzed workforce data of **1,470 employees**
- Measured employee attrition and retention trends
- Designed KPI cards for quick performance monitoring
- Created dynamic charts with slicers for easy analysis

---

## 🛠️ Tech Stack

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📂 Data Description

The dashboard is based on an HR dataset containing employee information such as:

- Employee Demographics
- Department
- Job Role
- Education
- Gender
- Age
- Attrition Status
- Job Satisfaction
- Monthly Income
- Years at Company

---

## 📈 Dashboard Metrics

| KPI | Description |
|------|-------------|
| Total Employees | Overall workforce count |
| Active Employees | Employees currently working |
| Attrition Count | Employees who left |
| Attrition Rate | Percentage of employee turnover |
| Average Age | Average workforce age |
| Job Satisfaction | Employee satisfaction score |

---

## 🔍 Business Insights

- Identified departments with higher employee turnover.
- Compared attrition across different age groups.
- Evaluated employee satisfaction levels.
- Examined workforce distribution by gender and education.
- Highlighted job roles with the highest attrition.
